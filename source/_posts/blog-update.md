4. 使这个Blog的全部文章更符合Markdown语法规范，例如（`{% note info %}`是[NexT主题的标签](https://theme-next.js.org/docs/tag-plugins/note)）：
    修改前：

    ```md
    ## 标题
    文本
    {% note info %}
    说明
    {% endnote %}
    ```

    修改后：

    ```md
    ## 标题

    文本

    {% note info %}
    说明
    {% endnote %}
    ```

由于这次更新涉及到许多方面，这次的更新应该算是这个Blog的比较大的更新了，~~虽然这个Blog还是很少人访问~~
